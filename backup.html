Raspberry Pi Comprehensive Backup Script

Simple Script to Create Backup of Critical Services

This guide demonstrates a simple method to create a comprehensive back up of a Raspberry Pi system. If placed within a .sh and ran, it will prompt the user to specify a writable backup location, validate input and safely copy critical files and directories such as; Pi-hole configurations, firewall rules (UFW) if exist, systemd services, network settings and Pi-hole's Teleporter export. This backup is then compressed into an archive with a file documenting its contents and system details to assist with easy restoration or migration.
This is a good first step to take, prior to making changes to your system such as utilizing this script to install Unbound for Fully Recursive DNS Resolution in order to increase the privacy and security of your DNS queries. 


#!/bin/bash
#This script allows you to create a backup prior to implementing Unbound, in case you
#want to revert to known settings/configurations. 
#This stores pihole settings, UFW settings, systemd, resolver & network config. 

# Pi-hole + system rollback snapshot script with user prompt for backup location
# Use with: nohup ./backup-pihole-backup.sh > ~/backup.log 2>&1 &

set -euo pipefail

# Root check
if [ "$(id -u)" -ne 0 ]; then
  echo "[ERROR] Run as root or with sudo"
  exit 1
fi

# Prompt user for backup directory
read -rp "Enter full path to backup directory: " BACKUP_ROOT

# Validate directory exists and is writable
if [ ! -d "$BACKUP_ROOT" ]; then
  echo "[ERROR] Directory does not exist: $BACKUP_ROOT"
  exit 1
fi

if [ ! -w "$BACKUP_ROOT" ]; then
  echo "[ERROR] Directory is not writable: $BACKUP_ROOT"
  exit 1
fi

TIMESTAMP=$(date +"%Y-%m-%d_%H-%M-%S")
BACKUP_DIR="$BACKUP_ROOT/snapshot-$TIMESTAMP"

echo "[INFO] Creating backup directory: $BACKUP_DIR"
mkdir -p "$BACKUP_DIR"

echo "[INFO] Backing up Pi-hole configuration..."
nice -n 10 ionice -c3 cp -a /etc/pihole "$BACKUP_DIR/"
sleep 1
nice -n 10 ionice -c3 cp -a /etc/dnsmasq.d "$BACKUP_DIR/"
sleep 1

echo "[INFO] Backing up UFW configuration..."
nice -n 10 ionice -c3 cp -a /etc/ufw "$BACKUP_DIR/"
sleep 1

echo "[INFO] Backing up systemd service files..."
nice -n 10 ionice -c3 cp -a /etc/systemd/system "$BACKUP_DIR/systemd-system"
sleep 1

echo "[INFO] Backing up resolver and network configs..."
nice -n 10 ionice -c3 cp -a /etc/resolv.conf "$BACKUP_DIR/"
sleep 1
if [ -d /etc/network ]; then
  nice -n 10 ionice -c3 cp -a /etc/network "$BACKUP_DIR/"
  sleep 1
fi
if [ -d /etc/NetworkManager ]; then
  nice -n 10 ionice -c3 cp -a /etc/NetworkManager "$BACKUP_DIR/"
  sleep 1
fi

echo "[INFO] Exporting Pi-hole Teleporter backup..."
nice -n 10 ionice -c3 pihole -a -t "$BACKUP_DIR/pihole-teleporter.tar.gz"
sleep 1

echo "[INFO] Writing manifest..."
cat < "$BACKUP_DIR/MANIFEST.txt"
Backup created: $TIMESTAMP
Hostname: $(hostname)
OS: $(uname -a)

Included:
- /etc/pihole
- /etc/dnsmasq.d
- /etc/ufw
- /etc/systemd/system
- /etc/resolv.conf
- network configuration (if present)
- Pi-hole Teleporter export
EOF

echo "[INFO] Creating compressed archive..."
nice -n 10 ionice -c3 tar -czf "$BACKUP_DIR.tar.gz" -C "$BACKUP_ROOT" "snapshot-$TIMESTAMP"

echo "[SUCCESS] Snapshot complete"
echo "Backup archive:"
echo "  $BACKUP_DIR.tar.gz"
