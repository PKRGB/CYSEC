Network IDS Setup with Suricata

Raspberry Pi OS Configuration, IDS install and Configuration & Security Hardening

This project demonstrates the complete installation and configuration of Suricata on Raspberry Pi OS, providing a clear, step-by-step guide for users of any experience level with steps outlined from preparing a blank SD card and installing the OS, to securely configuring and validating Suricata for network monitoring. Note: This guide reflects the constraints of the hardware which was on hand and thus used, specifically a Raspberry Pi Zero 2 W. For instance, despite implementing a swap file to increase memory size, Fail2Ban could not be installed due to CPU limitations but on newer, stronger models, installing Fail2Ban is a useful hardening tactic to protect against SSH login-abuse. A Readme specifically regarding System Hardening, including Fail2Ban, can be found here.
